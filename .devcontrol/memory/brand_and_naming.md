# SP-DevControl - Marca y Naming

## Nombre oficial
- Marca: SP-DevControl
- App local: SP-DevControl Local
- Binario recomendado: sp-devcontrol

## Naming tecnico estable
- Carpeta local: .devcontrol
- Config principal: devcontrol.config.json

## Regla
- No introducir nuevos artefactos publicos bajo la marca DevSentinel.
- Todo material nuevo debe usar la marca SP-DevControl.
